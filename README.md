# futurator-skills

Canonical **skill federation source** for the Futurator Labs pipeline.

This repo is the registry-of-record that `SKILL-SCOUT` and the Futurator-Admin
skills module resolve against. It is referenced by the daemon's
`~/.futurator/skill-federation.yaml` (priority 1, auto-trust).

## What's here

- **`index.json`** — the catalog of every skill the pipeline knows about, in the
  shape the resolver expects (`{ skills: [{ name, kind, version, license, description, ... }] }`).
  Generated by `scripts/gen-skill-index.mjs` in Futurator-Admin.

## Provenance

Skills are catalogued here for discovery, collision-checking, and the admin
catalog UI. Bodies are materialized by their origin:

- `provenance: bmad-method` — dev-workflow skills shipped by the `bmad-method`
  npm package (MIT), installed at project bootstrap. **56 skills.**
- `provenance: anthropics/skills` — Anthropic example skills. **3 skills.**

Futurator-authored skills (added via the admin authoring flow) will carry their
bodies in `skills/<name>/` here.

## Updating the catalog

Regenerate from a reference skill set:

```bash
node scripts/gen-skill-index.mjs --dir <skills-dir> --out index.json
```

See `docs/concepts/skills-management/skills-management-plan.md` in Futurator-Admin.
